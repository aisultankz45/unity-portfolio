# Project 1 — Lock Opening Game

**Жанр:** Puzzle / Logic  
**Платформа:** PC  
**Unity:** 2022.x

## Описание
Учебный проект: игра на логику с открытием замка.  
Игрок взаимодействует с элементами и решает задачу для открытия замка.

## Что сделал я
- Реализовал игровую логику на C#
- Работа с триггерами и коллайдерами
- Реакция объектов на действия игрока
- Настройка UI
